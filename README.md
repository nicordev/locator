# Locator

A website to draw things on a map, [here](http://locator.nicordev.com/index.html).

## Current features

* Show a map using [Leaflet](https://leafletjs.com/) and [MapBox](https://www.mapbox.com/)
* Draw circles
* Place markers
* Show popups when clicking on markers
* Geolocation
